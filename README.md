# 🚀 HypeSquad Manager

A clean and simple Python tool to manage your Discord **HypeSquad membership** directly from the terminal.

Join, leave, or check your current HypeSquad house with ease ⚡

---

## ⚠️ Disclaimer

> This project is for **educational purposes only**.

* This tool uses your **Discord user token**.
* Automating user accounts (**self-bots**) violates Discord Terms of Service.
* Your account **may get suspended or banned**.
* You are fully responsible for how you use this tool.

⚠️ **DO NOT use your main account.**

---

## ✨ Features

* 🔑 Set and store your Discord token
* 🏠 Join a HypeSquad house:

  * Bravery
  * Brilliance
  * Balance
* 🚪 Leave HypeSquad anytime
* 🔍 Check your current HypeSquad status
* 🎨 Clean CLI interface with colored output

---

## 📦 Requirements

* Python **3.6+**
* `requests`
* `colorama`

---

## 🔧 Installation

```bash
pip install requests colorama
```

Clone or download the repository, then navigate to the project folder.

---

## 🚀 Usage

Run the script:

```bash
python HypeSquadManager.py
```

### Main Menu

```
[1] > Set Token
[2] > Join HypeSquad
[3] > Leave HypeSquad
[4] > Check Status
[0] > Exit
```

---

## 🔐 Getting Your Token (Use Carefully!)

1. Open Discord in your browser
2. Press **F12** (Developer Tools)
3. Go to **Application → Local Storage → https://discord.com**
4. Copy the value of `token`

⚠️ **Never share your token with anyone**
